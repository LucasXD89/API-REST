# Senati Store - API REST

Este proyecto es una API RESTful construida para gestionar los datos de una tienda ("Senati Store"). Proporciona endpoints básicos para la interacción con la base de datos y está desarrollada utilizando el entorno Node.js.

## 🚀 Tecnologías Utilizadas

* **Node.js**: Entorno de ejecución para JavaScript.
* **Express.js**: Framework para construir la API y manejar las rutas.
* **MongoDB**: Base de datos NoSQL para almacenar la información (Compatible con local o MongoDB Atlas).
* **Mongoose**: Modelado de objetos de MongoDB para Node.js.

## 📋 Requisitos Previos

Antes de ejecutar este proyecto, asegúrate de tener instalado en tu sistema local:

* [Node.js](https://nodejs.org/) (Versión 14 o superior recomendada)
* [MongoDB](https://www.mongodb.com/) (Instalado localmente o tener una cuenta en MongoDB Atlas)
* [Git](https://git-scm.com/)
